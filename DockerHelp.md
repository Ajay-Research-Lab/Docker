**Get version of docker**

`docker -- version`

**Get help**

`docker container --help`

**Get list of images**

`docker image ls`

**Get list of containers**

`docker container ls --all` or `docker ps`

**To get a copy from docker hub**

`./get_image.sh`

**Running the docker container**

`./run_container.sh`

**Run a container with a particular command**

`./run_command.sh`
